Cyclops
========================

Cyclic coordinate descent for logistic, Poisson and survival analysis 

Getting Started
===============

1. In R, use the following commands to install Cyclops:

  ```r
  install.packages("devtools")
  library(devtools)
  install_github("ohdsi/Cyclops") 
  ```
